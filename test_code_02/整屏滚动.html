风铃
北漠公主，前任北王洪都的独生女。自幼从父习武，武艺高强。聪明伶俐又能歌善舞，曾以一曲风蝶之舞名扬天下。 在父亲北王洪都的影响之下，风玲自幼就展露出过人的领袖天赋，在北漠极具人气，深得民心。北漠的百姓都称她为“北漠最珍贵的风翡翠”。他们也一致认为，将来风玲会成为北漠的第一个女王。

天剑·南风
来自遥远南方大陆的神秘高人，其独创的南门功可以在数米之外震碎敌人的五脏六腑。居于南山之上，每日饮酒作诗，不问世事。有无数的高手登上南山向其挑战，无不狼狈而归，也有无数人慕名登山求艺 ，最终皆无功而返。 他身负一柄巨大石剑，却无人见其使用过，人称‘天剑’。

燃焰玫瑰—云娜
近年来在大陆上声名鹊起的一位刺客，武艺高强且精于用毒，暗杀时神出鬼没，从不失手。 传闻中的她是一名拥有着红发赤眼的美貌少女，好事的人们都希望着能一睹其真容，却又唯恐避之不及—就像那种盛放在大陆西岸，美丽却又致命的毒花—燃焰玫瑰

童年时期是一个孤苦伶仃的小女孩，在一次机缘中被鬼刀救过。后苦练武功成为杀手，希望可以回报鬼刀。 性格和冰公主截然相反，有着缜密的思维和更理性的判断，对待敌人手段残忍狠辣，绝不手软。暗杀术强于风铃 平时性格比较外向，有些狂妄又出言不逊，其实内心洞察一切，任何时候都不会放松警惕，极善心理战，对机会的把握也是非常娴熟，必要时甚至会自曝弱点来引诱敌人对其放松警惕。
